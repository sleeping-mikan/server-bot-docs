server-bot ドキュメント

server-bot は、Minecraft(統合版 / Java 版)などのゲームサーバーを Discord から起動・停止・操作・バックアップできる bot です。 サーバーログの Discord への転送、Web ブラウザからの操作、拡張機能による機能追加にも対応しています。

クイックスタート

  1. サーバー本体(server.exe / server.jar など)と同じ階層に server.py を配置する
  2. pip install -r requirements.txt で依存ライブラリをインストールする
  3. server.py を一度実行し、生成された .token に Discord bot のトークンを記述する
  4. .config の server_path / server_name を確認・修正する
  5. 再度 server.py を実行する — サーバー本体は bot が自動で起動します

詳しい手順と注意点は 導入ガイド を参照してください。

権限レベルについて

各コマンドには必要権限レベル(0〜5)が設定されており、 実行にはそのレベル以上の権限が必要です。 ユーザーの権限は /permission change で変更でき、必要レベル自体も .config の commands_level で調整できます。 初期状態では Discord サーバーの管理者権限を持つユーザーが最高権限として扱われます。
