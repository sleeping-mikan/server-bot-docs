# server-bot docs デザインシステム

このサイトは**ビルドなしの静的 HTML + 共有 CSS 1 枚**(`assets/style.css`)で構成される。
フレームワーク・CDN・Web フォントは使わない。すべてのページが同じヘッダー / サイドバー / フッターの骨格を持つ。

## デザインの核(変えてはいけないもの)

- **鮮やかなオレンジが主役**。`--orange-500: #f97316` / `--orange-600: #ea580c` を基調とし、
  くすんだ・柔らかいトーンへの全面的な置き換えは過去に差し戻された(NG)。
- **背景だけは温かいクリーム色** `--bg: #fdf6ec`。ここは彩度を上げない。
- **ライトモードのみ**。ダークモード対応はしない。
- カードに上縁のアクセントバーを付けない(過去に削除済み・NG)。
- コールアウト(`.note` / `.warn`)に左縁の太いアクセントボーダーを付けない(過去に削除済み・NG)。
  枠線は四辺とも同じ 1px。
- `--header-h` はヘッダーの実高さと、サイドバーの sticky `top` の両方で使う。
  片方だけ変えるとスクロール開始時に目次がズレて見える。必ずセットで扱う。

## カラートークン(:root)

| 用途 | 変数 | 値 |
|---|---|---|
| 主張色 | `--orange-500` / `--orange-600` | `#f97316` / `#ea580c` |
| 淡い面 | `--orange-50` / `--orange-100` | hover 背景・zebra・チップ |
| ページ背景 | `--bg` | `#fdf6ec`(温かいクリーム) |
| パネル | `--bg-panel` | `#ffffff` |
| コード背景 | `--bg-code` | `#fff3e6` |
| 本文 / 補足 | `--text` / `--text-sub` | `#2d2a26` / `#6b6357` |
| 罫線 | `--border` | `#f0e2d0` |
| リンク | `--link` | `--orange-700` |
| 警告系 | rose 系(`#be123c` など) | warn コールアウト専用。多用しない |

新しい色を足すときは必ず `:root` にトークンとして追加し、生の色コードを散らさない。

## タイポグラフィ

- フォント: OS 標準スタック(Segoe UI / Hiragino Sans / Noto Sans JP / Meiryo)。Web フォント禁止。
- 本文 `line-height: 1.75`。日本語ドキュメントなので詰めすぎない。
- 見出し階層は 3 段まで:
  - `h1` — ページタイトル。オレンジのグラデーション下線。1 ページに 1 つ。
  - `h2` — セクション。左端の丸いオレンジバー。必ず `id` を振る(ページ内リンク用)。
  - `h3` — 小見出し。`--orange-700` の文字色のみ。
- 和文と欧文・数字の間には半角スペースを入れる(例: `Python 3.12 の`)。

## コンポーネント一覧(class 名)

| class | 用途 |
|---|---|
| `.site-header` + `.inner` / `.logo` / `.gh-link` | 全ページ共通ヘッダー(オレンジのグラデーション) |
| `.layout` > `.sidebar` + `main` | 2 カラム骨格。sidebar は sticky |
| `.sidebar .group-title` / `a.active` | ナビ。現在ページに `class="active"` |
| `p.lead` | ページ冒頭のリード文 |
| `.note` / `.warn` | コールアウト。NOTE=オレンジ、WARNING=ローズ。アイコンは CSS の data URI |
| `.table-wrap > table` | 表は必ず `.table-wrap` で包む(横スクロール対応) |
| `.card-grid > .card` | index の入口カード。MDI アイコン + タイトル + 説明 |
| `.steps` | 番号付き手順。丸数字 + 縦の接続線 |
| `.search-wrap + .filter-box` / `.filter-count` | 絞り込み検索(commands / config) |
| `.lv` | 権限レベルバッジ |
| `.tips` | 補足の小さめテキスト |
| `.mdi` | インライン SVG アイコン(Material Design Icons のパスを直接埋め込む) |

アイコンは MDI の 24x24 パスを `<svg class="mdi" viewBox="0 0 24 24"><path d="..."/></svg>` として直接埋め込む。画像ファイルや icon font は使わない。

## 品質の作法

- 影は控えめに。`--shadow`(通常時)と hover 時の 2 段階まで。派手なグローは使わない。
- 角丸: 小物 6–8px、カード・表・コードブロック 10–12px。
- インタラクション: hover / focus には 0.12–0.2s の transition を付ける。
  `:focus-visible` に必ず見えるフォーカスリングを付ける(キーボード操作対応)。
- `prefers-reduced-motion` ではアニメーションを止める。
- モバイル(`max-width: 840px`)ではサイドバーがチップ状の横並びナビに変形する。
  ナビ項目を増やしたらモバイル表示も確認する。

## 新規ページの追加手順

1. 既存ページ(例: `web.html`)を複製し、`<title>` を「ページ名 | server-bot ドキュメント」にする。
2. 全ページの `.sidebar` にリンクを追加し、新ページでは自分に `class="active"` を付ける。
   サイドバーは全ページにコピーされているので、**全 HTML を漏れなく更新**する。
3. `h1` は 1 つ、リード文は `p.lead`。セクション `h2` には `id` を振る。
4. スタイルはインラインで書かず `assets/style.css` に追加する。
