拡張機能

server-bot は拡張機能によって Discord コマンド等を追加できます。

配置場所

server.py と同じディレクトリの mikanassets/extension に配置してください。

mikanassets/
└── extension/
    └── 拡張機能名/
        └── 拡張機能を実装したファイル群

拡張機能の実装仕様は server-bot-extensions で確認できます。

v3 での API 変更点

v3 では拡張機能 API のインポート方法が変更されています。v2 向けの拡張機能はそのままでは動作しません。 
# 旧 (v2)
from main import extension_commands_group, extension_logger

# 新 (v3)
from core.state import ctx
# ctx.extension_commands_group, ctx.extension_logger を使用する
