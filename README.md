# server-bot-docs

[server-bot-v3](https://github.com/sleeping-mikan/server-bot-v3) のドキュメントサイトです。

GitHub Pages で公開されます: https://sleeping-mikan.github.io/server-bot-docs/

## 構成

- `index.html` — ホーム(機能から探すカード / クイックスタート)
- `setup.html` — 導入ガイド(設置・実行・初期設定・Java 版)
- `commands.html` — コマンド一覧(検索フィルタ付き)
- `config.html` — .config リファレンス(検索フィルタ付き)
- `web.html` — Web コンソール
- `extensions.html` — 拡張機能
- `assets/style.css` — 共通スタイル(オレンジ基調・ダークモード対応)

ビルド不要の静的サイトです。ファイルを編集して push すれば反映されます。
