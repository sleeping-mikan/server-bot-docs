Web コンソール

Discord を経由せず、ブラウザからサーバーを操作・ファイルをダウンロードできる機能です。

アクセス方法

  1. Discord で /tokengen を実行し、ログイン用トークンを入手する
  2. ブラウザで http://ホストのIPアドレス:ポート番号 を開く(ポート番号は .config の web.port)
  3. ログイン画面でトークンを入力する
外部(インターネット側)からアクセスするには、ルーター等で追加のポート開放が必要です。

https で公開する場合(推奨)

https を用いて運用する場合は、リバースプロキシ(nginx、Cloudflare Tunnel など)を利用してください。

関連する設定

項目説明
web.portweb サーバーのポート番号。
web.secret_keyFlask のセッション鍵。十分に強固な文字列を設定してください。
web.use_front_pageweb ページからの操作を許可するか否か。false でもファイルダウンロードは可能です。
